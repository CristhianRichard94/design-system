# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.1] - Swap accent color from cyan to navy blue

### Changed

- Accent color scale (`--accent-*` tokens) switched from electric cyan to navy blue.

## [0.1.0] - Initial scaffold

### Added

- `@ds/tokens` — CSS custom properties + typed JS token object.
- `@ds/ui` — 8 core primitives (Button, Badge, Input, Card, Tooltip, Tabs, Dialog, DropdownMenu).
- `apps/showcase` — Next.js App Router app rendering every component's variants/states.
